# 🎓 EduPro Course Recommendation System

## Project Overview
The EduPro Course Recommendation System is a machine learning-based web application that recommends suitable courses to students based on their learning preferences and cluster segmentation.

## Features
- Student selection
- Personalized course recommendations
- Streamlit web interface
- Machine Learning (K-Means Clustering)
- Course recommendation dataset

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- NumPy
- Joblib

## Project Structure


## Installation

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
streamlit run app.py
```
